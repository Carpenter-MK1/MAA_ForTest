## 功能介绍

目前已有的功能：

- 指定**支援SSR数量**的刷初始

计划支持的功能：

* 指定任意种类SSR数量的刷初始
* 指定某一特定卡的刷初始
* 自动竞技场

画饼的功能：

* 全自动P卡

## 使用说明

下载地址：

### Windows

- 对于绝大部分用户，请下载 `MGPA-win-x86_64-vXXX.zip`
- 若确定自己的电脑是 arm 架构，请下载 `MGPA-win-aarch64-vXXX.zip`
- 解压后运行 `MaaPiCli.exe` 即可

### macOS

- 若使用 Intel 处理器，请下载 `MGPA-macos-x86_64-vXXX.zip`
- 若使用 M1, M2 等 arm 处理器，请下载 `MGPA-macos-aarch64-vXXX.zip`
- 使用方式：
  
  ```bash
  chmod a+x MaaPiCli
  ./MaaPiCli
  ```
